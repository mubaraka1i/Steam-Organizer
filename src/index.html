Help getting Steam API Key

Steam keys are created from your Steam account and are used to fetch profile and library data.

  1. Sign in to Steam and open the Steam API key signup page.
  2. Enter any domain name (ex. google.com), complete the registration (Will need to download the Steam Guard App), and copy the key that Steam shows you.
  3. Keep the key private. If you ever need a new one, revoke the old key and register again.
